# Конвертация SDF/SMILES в PDBQT
Скрипты для подготовки множества лигандов к докингу (в AutoDock Vina). 
- `generate_pdbqt.py` - с использованием RDKit и OpenBabel.
- `meeko_generate_pdbqt.py` - с использованием RDKit и Meeko. Поддерживает гибкость циклов (>5-членных).

## Основные возможности
1. Конвертация нескольких молекул, записанных в один файл: SDF/SMILES в PDBQT.
2. 3D-генерация: Автоматическое добавление водородов, конвертация в 3D, минимизация энергии.
3. Гибкость: Алифатические циклы > 5 атомов помечаются как гибкие.

## Требования
Установка необходимых библиотек:

```
pip install numpy scipy rdkit vina meeko gemmi autogrid openbabel-wheel
```
## Использование
1. Поместите исходные файлы SDF/SMI и скрипт в рабочую папку.
2. Запустите нужный вам скрипт:

```
python meeko_generate_pdbqt.py
# или
python generate_pdbqt.py
```
3. Введите полный путь к файлам прямо в консоль.
4. Результаты появятся в папке converted_pdbqts/.

## Настройки (в коде)
В объекте **prepper** класса **MoleculePreparation** можно изменить параметры гибкости:
`min_ring_size=6`: минимальный размер кольца, которое будет считаться гибким

## Пример вывода программы
В архиве `converted_pdbqts.zip` находятся сконвертированные молекулы из обоих входных файлов
